Website Poducts Quick View

Plugin is used to add quick view button in product sections (On Shop Page) on website, when customer will click on quick view button, A popup window is open. Customers can see product description on it and also select different variant. Customers can add to cart product directly from here.

Enable Option On Website

Go to website and enabled Quick View option from the Customize menu.
